# GrabPointXY
Grab X Y point's coordinate from image for manual segmentation 


Instructions
1. Click on the image to set the reference point (x=0, y=0).
2. Click to select points for the current boundary.
3. Press the SPACEBAR to save the current boundary and start a new one.
4. Press ENTER to finish the process and save all boundaries.
5. Each boundary will be saved to a separate text file.
